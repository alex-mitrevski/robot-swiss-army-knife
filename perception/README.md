# perception

A collection of components for various robot perception tasks.

The following ROS 2 packages are included here:
* `point_cloud_utils`: A C++-based package exposing point cloud-oriented functionalities. The package uses [PCL](https://pointclouds.org) for point cloud processing.
* `semantic_segmentation`: A Python-based package for semantic object segmentation from images. The package uses vision foundation models for object detection and segmentation.
